# Installation

<Steps>
<Step>

### Install the package

```sh
npm install @typedframe/core
```

</Step>
<Step>

### Install peer dependencies

```sh
npm install @mantine/core @mantine/form @tanstack/react-query zod axios @dnd-kit/core @dnd-kit/sortable
```

</Step>
<Step>

### Wrap your app

```tsx
import { QueryClient, QueryClientProvider } from '@tanstack/react-query';
import { MantineProvider } from '@mantine/core';

const queryClient = new QueryClient();

ReactDOM.createRoot(document.getElementById('root')!).render(
  <MantineProvider>
    <QueryClientProvider client={queryClient}>
      <App />
    </QueryClientProvider>
  </MantineProvider>
);
```

</Step>
<Step>

### Configure the API base URL (optional)

```ts
import { api } from '@typedframe/core';
api.defaults.baseURL = import.meta.env.VITE_API_BASE_URL ?? '/api';
```

</Step>
</Steps>

## Validate env vars with Zod

```ts filename="src/env.ts"
import { z } from 'zod';

const envSchema = z.object({
  VITE_API_BASE_URL: z.string().url().optional().default('/api'),
});

const parsed = envSchema.safeParse(import.meta.env);
if (!parsed.success) throw new Error('Invalid env vars');

export const env = parsed.data;
```

Import `./env` as the **first import** in `main.ts` so invalid configuration throws before any rendering begins.
