## Live demo app

The full end-to-end TaskBoard demo is deployed at:

**[typedframe-demo.vercel.app/examples/taskboard](https://typedframe-demo.vercel.app/examples/taskboard)**

It demonstrates:
- `SortableList` — drag rows to reorder tasks
- `useZodForm` — schema-backed live form editing
- `useAutosave` — changes persist after 600ms idle
- `useUpdateItem` — TanStack Query mutation with cache invalidation
